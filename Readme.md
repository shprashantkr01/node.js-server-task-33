This project displays my skill of building a server using node.js that serves html, css and javascript files.
__
Html, Css and js files used are from the previous Tasks in the course.
__

Steps to run the server :
step1: clone the repository from github using the command "git clone https://github.com/shprashantkr01/node.js-server-task-33.git"
step2: run the command "npm install" to install node modules.
step3: run the command "npm install nodemon" to install nodemon.
step4: run the command "npm run dev" to start the server.
step5: your server wiull be live at "http://localhost:3000".
